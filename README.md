# Vacation-Planer
## Summary

Det kan vara utmanande och tidskrävande för barnfamiljer att hitta aktiviteter och planera semestern som passar hela familjen. Med Vacation planer kan man få stöd i att planera semestrar utifrån antal familjemedlemmar och barnens ålder, önskemål, reseperiod och budget. AI analyserar informationen och kommer med förslag vilket kan ge inspiration. Data om flygpriser, familjevänliga hotell, aktiviteter på resemålet, väder och användarnas preferenser skulle användas. Bygga AI-projekt.
